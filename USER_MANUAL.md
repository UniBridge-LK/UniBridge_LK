## Getting Started

### First Launch

When you open UniBridge LK for the first time, you'll see the **Welcome Screen**. Here you have two options:

- **Sign Up**: Create a new account
- **Log In**: Sign in if you already have an account

### Sign Up Process

1. **Enter Your Email**
   - Use your school/university email for verification
   - Make sure it's a valid email address

2. **Create a Password**
   - At least 8 characters recommended
   - Include letters, numbers, and symbols for security

3. **Verify OTP**
   - Check your email for a one-time password (OTP)
   - Enter the 6-digit code in the app
   - Code expires in 10 minutes

4. **Complete Your Profile**
   - **Display Name**: Your full name or preferred name
   - **University**: Select your university from the list
   - **Faculty**: Choose your faculty
   - **Department**: Select your department
   - **Account Type**: Student, Staff, or Alumni
   - **Profile Photo** (optional): Upload a profile picture

5. **Review & Confirm**
   - Review your information
   - Tap "Create Account"
   - You're all set!

### Log In

1. Enter your registered email
2. Enter your password
3. Tap "Login"
4. You'll be taken to the Home screen

### Forgot Password?

1. On the Login screen, tap "Forgot Password?"
2. Enter your registered email
3. Check your email for a password reset link
4. Follow the instructions to create a new password
5. Log in with your new password

## Account Management

### View Your Profile

1. Tap the **Profile** icon (bottom navigation bar, rightmost)
2. You'll see:
   - Your profile picture
   - Display name
   - University, Faculty, Department
   - Account type
   - Bio/About section

### Edit Your Profile

1. Go to **Profile**
2. Tap the **Edit** button (pencil icon)
3. Update any of the following:
   - Display name
   - University, Faculty, Department
   - Account type
   - Profile photo
   - Bio (if available)
4. Tap **Save Changes**

### Change Password

1. Go to **Profile**
2. Tap **Settings** (gear icon)
3. Select **Change Password**
4. Enter your current password
5. Enter your new password (confirm)
6. Tap **Update Password**

### Delete Account

⚠️ **Warning**: This action cannot be undone.

1. Go to **Profile** → **Settings**
2. Scroll to **Danger Zone**
3. Tap **Delete Account**
4. Confirm by typing your password
5. Your account and all associated data will be permanently deleted

## Finding & Connecting with People

### Browse People Directory

1. Tap the **People** icon (bottom navigation, second from left)
2. You'll see a list of all users in the system
3. Each card shows:
   - Profile picture
   - Display name
   - A button showing connection status

### Search for People

1. In the **People** view, use the search bar at the top
2. Type to filter by:
   - Name
   - University
   - Faculty
   - Department
3. Results update in real-time

### Send a Connection Request

1. Find the person you want to connect with
2. Tap the **Connect** button on their card
3. A dialog will appear with fields for:
   - **Add a note** (optional): Introduce yourself or explain why you want to connect (max 250 characters)
4. Tap **Send Request**
5. The button changes to **Request Sent**

### Accept Connection Requests

1. Go to **Chats** (bottom navigation, third from left)
2. Tap the **Requests** tab (second tab)
3. You'll see all pending connection requests
4. For each request, you can:
   - **Accept**: Tap the green **Accept** button to confirm the connection
   - **Decline**: Tap the red **Decline** button to reject
5. Accepted connections appear in your **Chats** tab

### View Connected Users

1. Go to **Chats**
2. You'll see all your connected users listed
3. Connected users show a **Connected** button

### Remove a Connection

1. Open a chat with a connected user
2. Tap the **Options** menu (three dots, top right)
3. Select **Remove Connection**
4. Confirm the action
5. You are no longer connected (can reconnect later)

## Messaging

### Start a Chat

1. Go to **Chats** (bottom navigation)
2. Tap the **Chats** tab (first tab)
3. From the **People** view, tap a **Connected** user button to open chat
4. Alternatively, if you already have an active connection, they appear in your chats list

### Send a Message

1. Open a chat conversation
2. Type your message in the text field at the bottom
3. Tap the **Send** button (paper plane icon)
4. Your message appears with a **pending** status
5. Once received by the server, status changes to **sent**
6. When the recipient views it, status changes to **seen**

### View Message Timestamps

- Hover or long-press on any message to see the exact time it was sent
- Times display in 12-hour format (e.g., 2:30 PM)

### Edit a Message

1. Long-press on your own message (not on received messages)
2. Tap **Edit Message**
3. Modify the text in the dialog
4. Tap **Save**
5. The message is updated for both you and the recipient
6. An **edited** tag appears next to the timestamp

### Delete a Message

1. Long-press on your own message
2. Tap **Delete Message**
3. Confirm the deletion
4. The message is replaced with "Message deleted" for both parties
5. Recipient sees the deletion

### Mark Conversation as Read

- Messages automatically mark as read when you open a chat
- Unread badges disappear from the **Chats** tab

### Notification Badge

- A red badge with a number on the **Chats** tab icon shows unread message count
- Badges for individual chats show in the conversations list

### Offline Messaging

- If you're offline, messages are saved locally
- A "Syncing..." indicator appears at the top
- Messages sync when you reconnect to the internet
- Offline messages show **pending** status until synced

## Events

### Discover Events

1. Tap the **Events** icon (bottom navigation, fourth from left)
2. Browse all upcoming events
3. Each event card shows:
   - Event title
   - Date and time
   - Location or platform (for online events)
   - Host organization
   - Category (Workshop, Seminar, etc.)

### Search Events

1. In the **Events** view, use the search bar
2. Filter by:
   - Event name
   - University
   - Faculty
   - Department
3. Results update instantly

### View Event Details

1. Tap **View Details** on any event card
2. You'll see:
   - Event title and description
   - Date, time, and location
   - Host organization and organizer details
   - Event type (Physical or Online)
   - Platform (for online events, e.g., Zoom, Google Meet)
   - Attendee count
   - "About this event" section with full description

### Attend an Event

1. Open event details
2. Tap **Mark as Attending** (if available)
3. Event appears in your "My Events" section
4. You receive notifications for event updates

### Create an Event

1. Go to **Events**
2. Tap the **+** button (floating action button, bottom right)
3. Fill in the event details:
   - **Event Title** (required)
   - **Description** (optional)
   - **Date & Time** (required)
   - **Location** or **Platform** (depends on event type)
   - **Event Type**: Physical or Online
   - **Category**: Select from dropdown
   - **Host Organization** (optional)
4. Tap **Create Event**
5. Event is published and visible to all users

### Edit an Event

1. Open event details
2. Tap the **Edit** button (pencil icon, top right)
3. Only event creators can edit
4. Modify details and tap **Save**

### Delete an Event

1. Open event details
2. Tap the **Options** menu (three dots)
3. Select **Delete Event**
4. Confirm deletion
5. Event is removed from the system

## Forums

### Access Forums

1. Tap the **Home** icon (bottom navigation, first)
2. Scroll down to find the forum section
3. You'll see different forum categories:
   - **Global Forum**: Discussions across all universities
   - **University Forums**: Specific to each university
   - **Faculty Forums**: Faculty-level discussions
   - **Department Forums**: Department-specific threads

### Browse Discussions

1. Select a forum category
2. You'll see all discussion threads
3. Each thread shows:
   - Thread title
   - Author name
   - Number of replies
   - Last updated time

### Create a Discussion Thread

1. In any forum, tap **Create Thread**
2. Enter:
   - **Title**: Brief topic description
   - **Content**: Your question or discussion point (supports text formatting)
   - **Tags** (optional): Add relevant tags for discoverability
3. Tap **Post Thread**
4. Your thread appears at the top of the forum

### Reply to a Thread

1. Open a discussion thread
2. Scroll to see all existing replies
3. At the bottom, type your reply in the text field
4. Tap **Post Reply**
5. Your reply appears instantly

### Edit a Post

1. Hover/long-press on your post
2. Tap **Edit**
3. Modify the content
4. Tap **Save**

### Delete a Post

1. Long-press on your post
2. Tap **Delete**
3. Confirm deletion
4. Post is removed from the thread

### Like/Vote on Posts

1. On any post or reply, tap the **Like** icon (heart)
2. Post counter increases
3. Tap again to unlike

### Mark Thread as Solved

- Thread creators can mark a reply as the solution
- Tap **Mark as Solution** on helpful replies
- Solution appears highlighted with a checkmark

## Troubleshooting

### I can't log in

**Problem**: "Invalid email or password" error

**Solutions**:
1. Check spelling of email address
2. Verify CAPS LOCK is off
3. Try the "Forgot Password?" option
4. Ensure your account was created successfully
5. Contact support if the issue persists

### I'm not receiving messages

**Problem**: Messages aren't appearing in chats

**Solutions**:
1. Check your internet connection (WiFi or mobile data)
2. Pull down to refresh the chat list
3. Close and reopen the app
4. Check if you're blocked by the sender
5. Restart your phone

### My message says "pending" for too long

**Problem**: Message stuck in pending state

**Solutions**:
1. Check internet connection
2. Go to settings and check sync status
3. Tap "Retry" if available
4. Force close the app and restart
5. Reconnect to WiFi/mobile network

### I can't find a user

**Problem**: User doesn't appear in search results

**Solutions**:
1. Verify the spelling of their name
2. Try searching by university/faculty instead
3. User may have deleted their account
4. User may have changed their display name
5. Try searching again after a few minutes

### Can't create an event

**Problem**: "Premium feature" message appears

**Solutions**:
1. UniBridge LK currently has free event creation
2. Contact support if you see this unexpectedly
3. Ensure you're logged in with correct account
4. Try refreshing the app

### Connection request not sending

**Problem**: Request fails or shows error

**Solutions**:
1. Check internet connection
2. Ensure you haven't already sent a request to this user
3. Check if you're not blocked by the user
4. Restart the app
5. Try again after a few minutes

### Offline mode not working

**Problem**: Messages aren't syncing when back online

**Solutions**:
1. Check stable internet connection
2. Go to Chats and look for sync indicator
3. Tap "Retry" button if shown
4. Close and reopen the app
5. Check phone storage (may need space for cache)

## FAQs

### General

**Q: Is UniBridge LK free to use?**
A: Yes, UniBridge LK is completely free. No paid subscriptions or premium tiers are currently required.

**Q: What's the minimum age to join?**
A: You must be at least 13 years old to create an account. We recommend users under 18 to have parental consent.

**Q: Can I have multiple accounts?**
A: Each person should have one account. Multiple accounts are not permitted and may result in account suspension.

### Account & Privacy

**Q: Is my personal information secure?**
A: Yes. We use industry-standard encryption (Firebase) and follow strict privacy policies. Your data is never shared with third parties without consent.

**Q: Can I make my profile private?**
A: Currently, all profiles are visible to other authenticated users. Privacy controls may be added in future updates.

**Q: Can I delete my account?**
A: Yes, you can permanently delete your account in Settings. **Warning**: This action cannot be undone and all your data will be removed.

**Q: What happens to my messages if I delete my account?**
A: Your messages will remain in conversations but will appear as "Deleted User."

### Messaging

**Q: Can I unsend a message?**
A: No, but you can edit or delete messages. Deleted messages show as "Message deleted" for both parties.

**Q: Can I send messages to non-connections?**
A: Not at the moment. You must be connected first.

**Q: How long are messages stored?**
A: Messages are stored indefinitely in your cloud backup and until you delete your account.

**Q: Can I send images or files?**
A: Currently, text-only messaging is supported. Image and file sharing may be added in future updates.

### Connections

**Q: How many connections can I have?**
A: There's no limit to the number of connections you can have.

**Q: Can I block someone?**
A: Yes, go to Settings and use the Block Users feature.

**Q: What happens if I block someone?**
A: The blocked user won't be able to message you, see your profile, or send connection requests.

**Q: Can I unblock someone?**
A: Yes, go to Settings and remove them from your blocked list.

### Events

**Q: Can I create events as a student?**
A: Yes, students, staff, and alumni can all create events.

**Q: How many people can attend an event?**
A: There's no limit to event attendance.

**Q: Can I edit an event after creating it?**
A: Yes, only the event creator can edit event details.

**Q: Can I see who's attending an event?**
A: Yes, view the attendee list in event details.

### Forums

**Q: Can anyone post in forums?**
A: Yes, any authenticated user can create threads and post replies in any forum.

**Q: How do I report offensive content?**
A: Use the "Report" button on any post. Our moderation team will review.

**Q: Can I delete my forum posts?**
A: Yes, you can delete your own posts anytime.

**Q: Are forum posts permanent?**
A: Posts remain visible unless you delete them or are reported and removed by moderators.

### Technical Issues

**Q: The app crashes frequently. What should I do?**
A: 
1. Try clearing app cache (Settings → Apps → UniBridge LK → Clear Cache)
2. Uninstall and reinstall the app
3. Ensure you're running the latest version
4. Contact support with crash details

**Q: How do I report a bug?**
A: Go to Settings → Report a Bug and describe the issue with screenshots if possible.

**Q: Why is the app running slowly?**
A:
1. Check internet speed
2. Close other apps
3. Clear app cache
4. Restart your phone
5. Update to latest app version

**Q: I'm getting "offline" messages. What does it mean?**
A: Your internet connection is unstable. Connect to WiFi or move to an area with better signal. Messages will sync automatically when you're back online.
